# Model screenshots — drop your real TradingView charts here

The Manual tab loads one image per model. Save your (or Anbu's) TradingView
screenshots here with these EXACT names (.jpg or .png both work — if you use
.png, the app's onerror still shows a placeholder unless you also rename the
src, so prefer .jpg):

- 930-manipulation.jpg   → the 9:30 Manipulation (read-only) example
- 930-expansion.jpg      → the 9:30 Expansion (retrace-into-FVG) example
- 1000-reversal.jpg      → the 10:00 AM Reversal (main model) example
- 1000-continuation.jpg  → the 10:00 AM Continuation example
- inversion-fvg.jpg      → an inversion-FVG entry example
- amd.jpg                → an AMD (Accumulation-Manipulation-Distribution) day profile

Until a file exists, the Manual shows a labelled placeholder box for that slot.
No AI-drawn charts — only real screenshots go here.
